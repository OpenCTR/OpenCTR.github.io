OpenCTR Sub-Projects

Here is a list of all the current sub-projects that are a part of OpenCTR:

https://github.com/OpenCTR/OpenCTR
OpenCTR SDK Builder.
https://github.com/OpenCTR/OpenCTR-Tools
OpenCTR Tools, including 3dsxtool and smdhtool.
https://github.com/OpenCTR/OpenCTR-Toolchain
CMake project to build the OpenCTR cross-compiler toolchain.
https://github.com/OpenCTR/OpenCTR-Doc
Source code of this very documentation.
https://github.com/OpenCTR/libctr
Software library for Nintendo 3DS homebrew.
